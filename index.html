String Manipulation

Strings in Python Are Zero Based

strVar = '123456789'

strVar[0]
return the first character of the variable
strVar[-2]
return the second to last character
strVar[:3]
return the first 3 characters
strVar[-3:]
return the last three characters
strVar[3:]
return all but the first three characters

Useful String Methods

label = 'US Hwy 40'

label.upper()
convert a string to uppercase
label.replace()
replace a substring within a string
label.strip()
remove a substring or spaces from the start or end of a string
label.split()
split a string into a list based on a split character(s)
label.endswith()
return True if the string endswith a specified value
'{}'.format()
concatenate elements into a string based on defined positions
